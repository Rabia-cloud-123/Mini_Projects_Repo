## How to Contribute
1. **Fork** this repository or get access from the owner.
2. Go to the folder you want to add your project in (or create a new folder inside `Games/`).
3. Add your project files. You can organize them in **subfolders** if needed.
4. Commit your changes and submit a **pull request** if you are using a fork.
